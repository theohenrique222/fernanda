# AGENTS.md

## Stack
- Vue 3
- TypeScript
- Vite
- Tailwind CSS

## Code Style
- Use Composition API
- Prefer `<script setup>`
- Components must be reusable
- Use TypeScript interfaces
- Avoid inline styles
- Use Tailwind utility classes

## Architecture
- Components in `/components`
- Pages in `/pages`
- Composables in `/composables`
- Services/API in `/services`

## UI Rules
- Modern UI/UX
- Responsive design
- Soft shadows
- Rounded corners
- Consistent spacing

## Naming
- Components: PascalCase
- Composables: useSomething
- Variables/functions: camelCase

## Important
- Do not break existing layouts
- Preserve responsiveness
- Prefer clean and minimal code
